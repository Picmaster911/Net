3. Нужно выполнить это на самом разбери
Чтобы упростить разрешение пути, добавьте переменную среды DOTNET_ROOT и добавьте каталог .dotnet к $PATH с помощью следующих команд:
Команды, которые вы привели, предназначены для установки переменных окружения, используемых .NET SDK,
и добавления их в файл .bashrc для того, чтобы они автоматически применялись каждый раз, когда вы открываете новую сессию терминала.
Вот как это работает:
Эта команда добавляет строку export DOTNET_ROOT=$HOME/.dotnet в конец файла .bashrc в вашем домашнем каталоге.
Файл .bashrc — это сценарий, который выполняется при каждом запуске интерактивного терминала Bash.
export DOTNET_ROOT=$HOME/.dotnet устанавливает переменную окружения DOTNET_ROOT в значение $HOME/.dotnet. 
Это указывает на путь, где установлен .NET SDK.
source ~/.bashrc
Эта команда применяется для немедленного применения изменений, внесенных в файл 


echo 'export DOTNET_ROOT=$HOME/.dotnet' >> ~/.bashrc
echo 'export PATH=$PATH:$HOME/.dotnet' >> ~/.bashrc
source ~/.bashrc

5.Переносим файлы с компа на arm64 по SSH
